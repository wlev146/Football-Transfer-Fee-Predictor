# Football-Transfer-Fee-Predictor
Built an end-to-end machine learning pipeline in Databricks to predict football transfer fees across 500 plus historical transactions spanning 2009 to 2026. Engineered an ensemble VotingRegressor combining Random Forest and Gradient Boosting with outlier handling and market value blending, achieving a RMSE of €8.6M on negotiated transfers. A coefficient-based fee evaluator with 12 interactive parameters including position, age, league, club, and contract length enables real-time estimates, while a 7-page interactive dashboard covers league economics, player demographics, contract timing analysis, and a 10-year market projection model validated against CIES Football Observatory research projecting 9% annual inflation across 5,244 transactions.
